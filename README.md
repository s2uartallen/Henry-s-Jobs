# 🎮 Henry's Jobs - Career Adventure Game

A colorful, engaging educational video game designed for 10-year-olds in Australia. Players explore 13 different occupations, complete job-related tasks, and earn virtual paychecks while learning about various careers and Australian facts.

## 🎯 Game Features

### 13 Occupations to Explore
1. **Doctor** 👨‍⚕️ - Learn about health and human biology
2. **Lawyer** 👨‍⚖️ - Discover legal knowledge and safety rules
3. **Firefighter** 👨‍🚒 - Master fire safety and emergency response
4. **Teacher** 👨‍🏫 - Test your knowledge across subjects
5. **Chef** 👨‍🍳 - Learn about food and cooking in Australia
6. **Astronaut** 👨‍🚀 - Explore space and planets
7. **Veterinarian** 👨‍⚕️🐾 - Care for animals and learn about native species
8. **Architect** 👷‍♂️ - Design buildings and structures
9. **Musician** 🎵 - Master music and Australian artists
10. **Farmer** 👨‍🌾 - Understand agriculture and farming
11. **Police Officer** 👮‍♂️ - Learn about community safety
12. **Pilot** 👨‍✈️ - Navigate the skies
13. **Game Designer** 👨‍💻🎮 - Create fun interactive experiences

### Game Mechanics
- 🎓 **Educational Tasks**: Multiple-choice questions related to each job
- 💰 **Paycheck System**: Earn virtual money for completing jobs
- 📈 **Level Progression**: Level up every 3 jobs completed
- 🔓 **Job Unlocking**: New jobs unlock as you progress
- 🎵 **Background Music**: Engaging audio for immersion
- 📱 **Responsive Design**: Works on desktop and mobile devices
- 💾 **Save System**: Progress is automatically saved to local storage

## 🎓 Educational Content
Each job includes:
- Real-world facts about the occupation in Australia
- 3 engaging multiple-choice questions
- Educational explanations for answers
- Australian context and examples

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs in browser

### Running the Game
1. Open `index.html` in your web browser
2. Click "Start Game" to begin
3. Select a job to complete tasks
4. Answer questions correctly to earn money
5. Level up to unlock new jobs

## 🎮 How to Play

1. **Main Menu**: Start the game or read instructions
2. **Job Selection**: Choose from available unlocked jobs
3. **Job Tasks**: Answer 3 questions for each job
4. **Earn Rewards**: Get paid and level up
5. **Unlock Jobs**: Complete 3 jobs to level up and unlock a new occupation
6. **Victory**: Unlock all 13 jobs to complete the game
